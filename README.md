# Installation
### 1.Clone this Project
```
git clone https://github.com/argon1025/Hamster-server.git
```

### 2.install all dependencies
```
npm install
yarn install
```

### 3.Started dev
```
npm run start
yarn start
```

# Build this solution
### `mac`
```
npm run build-mac
yarn build-mac
```
### `Window`
```
npm run build-win
yarn build-win
```
